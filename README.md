## Background

Visit the output page above to visualize what has been extracted.

## How-to

To run the scripts in this repository:
- install [nodejs v6+](https://nodejs.org/en/)
- install yarn

```shell
npm install yarn -g
```

- install dependencies

```shell
yarn install
```

- run the script

```shell
npm run scrape
```
